Project Description

The software runs some analysis on a molecular dynamics simulation of a protein. Users are able to query protein data and run their own analysis. The protein's data frame is stored in a database, and each frame can be searched for and examined. A graph of the protein, as well as a 3d graphic of it, are present on the interface.
